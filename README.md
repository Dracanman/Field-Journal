# The Vael Field Journal

A self-contained, offline-friendly creative kit for the Godling Universe.

This **public** repo hosts only the app shell (HTML + manifest + service worker +
icon) so it can be installed via GitHub Pages and added to an iPad home screen.
It contains **no private canon and no token** — your journal and canon live in a
separate private repository, reached through a fine-grained GitHub token you enter
in the app's **☁ My file** panel at runtime (stored only on your device).

**Use it:** open the Pages URL in a browser, or Add to Home Screen on iPad.
